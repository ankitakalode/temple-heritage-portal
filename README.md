🛕 Temple Management System

A complete web-based Temple Management System developed using PHP and MySQL. The project helps temples manage devotees, donations, pooja bookings, priests, and temple events through a centralized admin dashboard.

---

Features

- User Registration & Login
- Admin Dashboard
- Donation Management
- Pooja Booking
- Devotee Management
- Event Management
- Priest Management
- MySQL Database
- Responsive User Interface
- Secure Authentication

Installation

1. Install XAMPP.

2. Copy the project folder into the htdocs directory.

3. Start Apache and MySQL.

4. Import the SQL database into phpMyAdmin.

5. Open your browser.

6. Run:

http://localhost/Temple-Management-System

---

Database

Create a MySQL database and import the provided SQL file.

Main Tables:

- users
- donations
- bookings
- priests
- events
